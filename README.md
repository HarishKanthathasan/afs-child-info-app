# Alliance Finance
![Screenshot](public/Alliance%20Finance%20Company.png)

## Prerequisites

* **Node.js and npm:** Ensure you have them installed on your system. Download them from https://nodejs.org/en/.

## Installation

**Clone the repository:**
```bash
   git clone
```
**Install dependencies:**
```bash
   npm install
```
## Run backend code
```bash
   cd backend
   npm init -y
   node server.js
```

## Run frontend code
```bash
   npm start
```
